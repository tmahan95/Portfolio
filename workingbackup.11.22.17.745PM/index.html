Portfolio Introduction

This portfolio is a completely custom site hosted on a Ubuntu server. It is comprised of several sections, each with assignments and links to repositories demonstrating the skillset I have acquired in my education. There is a side menu for quickly jumping to the desired section.
To view an assignment, click the blue button associated with it. To close the document, click the button again.

This site relies on the PDF.js project from Mozilla and a lot of tutorials from W3 schools for uploading files and viewing PDFs on this site.

SystemsAnalysisAndDesign

This section is a collection of documents from a semester-long project, in which my team and I designed a system for the University of Louisville School of Nursing. The documentation included is mostly work from the deliverables at each phase of design. The approach taken was a bit unusual, as it involved elements of waterfall and agile development. Waterfall was included as we did no developement during this phase, we only designed, refined, and redesigned as needed.

Management of Information Systems

Managmeent of Informations Systems is centered around analyzing the current situation an organization is facing, and the alternatives available to the organization in each situation. The analysis in these cases includes Porter's Five Forces theory, as well as stakeholder analysis of each alternative available for selection. The skill analytical skills gained from these assignments have proven useful even in my current company for identifying the root cause of problems, and how to address them in the best manner for the organization.

NetworkSecurity

This section has assignments that deal with both theory and practice behind network and information security. The first group of ICE assignments deal mostly with information security, the later labs deal with network security and how to transport information safely across a network.
